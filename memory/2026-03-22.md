## Technical Decisions

### Why iOS is Different from macOS
- **iOS constrains:** No app blocking, no global event tapping, limited notifications
- **iOS pivot:** Made it a **focus timer + goal tracker** instead
- **Shared licensing:** Both use same `STILLMODE-XXXX-XXXX-XXXX` format
- **Cross-platform:** Users buy once, use on both

### Architecture Choices
- **macOS:** NSWindow layering for overlays (NSWindow.Level.screenSaver)
- **iOS:** SwiftUI full-screen with progress ring (native approach)
- **Data:** SwiftData for both (modern, local-first)
- **Licensing:** Keychain for both (device-encrypted)

### Why This Timing Works
- macOS is immediately launchable (code-signed + notarized can ship Monday)
- iOS development is ~1-2 weeks to App Store ready
- Both apps mature in parallel
- Users who buy macOS get iOS for free with same license key (ecosystem lock-in)
