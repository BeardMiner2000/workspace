## Still Mode App — Built & Launch Planning

### What was built
- macOS menubar focus app: StillMode.app
- Lives at: `/Users/jl/.openclaw/workspace/stillmode/`
- Built with Swift/AppKit, compiles clean with Xcode
- 🌙 menubar icon → click → pick one app → everything else hides + DND on → 🌕
- Exit: click again, all apps restore, DND off

### Business / Launch Plan (from Signal + webchat session)
- **Domain idea:** `stillmode.app`
- **Price:** $2.99
- **Model:** Direct sale (NOT App Store)
  - App Store = 30% cut ($0.90 gone on every $2.99 sale)
  - Direct = ~97% kept after payment processor fees
  - App Store also requires sandboxing which conflicts with Accessibility entitlements needed to hide other apps
- **Distribution:** Direct download + notarize for Gatekeeper
- **Storefront:** Gumroad or Lemon Squeezy (simplest for indie devs)
- **Unlock model:** Was discussed — simple one-time purchase, download the .app directly
- README explicitly documents App Store submission notes and why direct is better

### Tech notes
- Needs Accessibility permission to hide other apps (System Settings → Privacy → Accessibility)
- DND via `defaults write com.apple.notificationcenterui doNotDisturb` (works macOS 13/14; macOS 15 may need Focus framework)
- No external dependencies
- Built/tested on JL's MacBook Pro (arm64, macOS 26.x)
