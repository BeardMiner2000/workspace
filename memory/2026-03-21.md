## Key Learnings

1. **Agents work best for review, not implementation**
   - Use agents to: review code on disk, find bugs, suggest fixes
   - Don't use agents to: write code in isolation, claim success without verification

2. **Real testing is essential**
   - Automated menu interactions via cliclick revealed real functionality
   - Manual testing on actual macOS beats any simulation

3. **Menu handling is tricky**
   - Rebuilding menus while open causes state corruption
   - Solution: update items in-place instead of rebuild/close cycles

4. **Keychain for license storage is solid**
   - Simple, secure, user-friendly
   - Good foundation for freemium model
