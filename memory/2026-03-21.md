## Key Technical Insights

### The Real Problem with Agents
- Agents said they "fixed" code but never wrote files to disk
- I should have verified file contents immediately
- **Better process:** Spawn agent → verify files → have agent review actual files → iterate

### The Critical Menu Bug
The most important fix was realizing that `performClick(nil)` closes the menu. When rebuilding after selection:
1. User clicks app
2. Code schedules menu rebuild
3. `buildAndShowMenu()` assigns new menu and calls `performClick(nil)`
4. This closes the menu while user is trying to interact
5. Next click doesn't register because menu state is confused

**Solution:** Update menu items in-place using NSMenuItem state properties instead of rebuilding the whole menu.

### What Actually Works
