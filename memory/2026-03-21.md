## What Happened

## Lessons Learned

1. **Agent work verification:** Don't trust completion messages. Verify code actually exists on disk.
2. **Hybrid approach works:** Real human coding + agent code review > pure agent delegation
3. **Menu handling is tricky:** Race conditions happen when rebuilding UI in response to user interaction. Update in-place instead.
4. **Automated testing with cliclick:** Much better than manual testing. Can iterate fast.
5. **NSWorkspace API:** Works well for monitoring app activations. Notification center matters.
